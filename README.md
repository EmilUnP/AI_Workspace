# Eduator AI Workspace

Minimal, reset documentation for the current product baseline.

Current baseline version: **0.0.8**.

## Current Scope

- Web app (`apps/web-app`)
- API server and shared packages (`packages/*`)
- SaaS app is removed
- Student portal UI is removed and blocked

## Quick Start

```bash
npm install
npm run dev:api
npm run dev:web
```

## Main Scripts

- `npm run dev:all` - Start API and Web app together
- `npm run build:apps` - Build Web app
- `npm run lint` - Lint workspace
- `npm run type-check` - TypeScript check
